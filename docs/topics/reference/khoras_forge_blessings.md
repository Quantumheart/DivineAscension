# Khoras - God of the Forge & Craft

**Version:** 2.0.0 (Utility Redesign)
**Last Updated:** 2025-11-29
**Status:** Design Phase - Implementation In Progress

---

## Overview

**Domain:** Smithing, Tool-making, Durability
**Philosophy:** *"Mastery through craft. Survival through preparation."*

### Playstyle

Tool durability and ore efficiency focused. Bonuses to tool longevity and ore gathering. Melee damage to protect their companions.

### Earning Favor

- Mining ore: 2 favor per ore
- Smithing items: 5-15 favor per craft
- Smelting items: 1-8 favor
- PvP kills: Standard
- Passive: 0.5/hour

---

## Player Blessings (6 Total)

### Tier 1 - Initiate (0-499 favor)
**"Craftsman's Touch"**
- Tools last **+10%** longer
- **+10%** ore yield

---

### Tier 2 - Disciple (500-1999 favor) - Choose Path

**Path A: "Masterwork Tools"** (Utility)
- Tools last **+15%** longer (total: 25%)
- **+10%** mining speed

**Path B: "Forgeborn Endurance"** (Survival)
- **+10%** melee weapon damage 
- **+10%** max health

*Both require: Craftsman's Touch*

---

### Tier 3 - Zealot (2000-4999 favor) - Specialization

**Path A: "Legendary Smith"**
- Tools last **+20%** longer (total: 45%)
- **+15%** ore yield (total: 25%)

*Requires: Masterwork Tools*

**Path B: "Unyielding"**
- **+10%** reduced armor durability loss
- **+15%** max health (total: 25%)

*Requires: Forgeborn Endurance*

---

### Tier 4 - Champion (5000+ favor) - Capstone

**"Avatar of the Forge"**
- Tools repair **1 durability per 5 minutes**
- **10%** Armor walk speed

*Requires: Both Legendary Smith AND Unyielding*

---

## Religion Blessings (4 Total)

### Tier 1 - Fledgling (0-499 prestige)
**"Shared Workshop"**
- All members: Tools last **+10%** longer
- All members: **+10%** ore yield

### Tier 2 - Established (500-1999 prestige)
**"Guild of Smiths"**
- All members: Tools last **+15%** longer
- All members: **+15%** ore yield

### Tier 3 - Renowned (2000-4999 prestige)
**"Master Craftsmen"**
- All members: Tools last **+20%** longer
- All members: **+20%** ore yield
- **10%** Armor walk speed
### Tier 4 - Legendary (5000+ prestige)
**"Pantheon of Creation"**
- All members: **+10%** max health

## Combined Maximums (Player + Religion, Both Paths)

- **65% longer tool durability** (45% player + 20% religion)
- **45% more ore yield** (25% player + 20% religion)
- **35% max health** (25% player path B + 10% religion)
- **20% armor walk speed** (10% player capstone + 10% religion T3)
- **+10% melee weapon damage** (player path B)
- **10% reduced armor durability loss** (player path B)
- Passive tool repair (1/5min player)

---

## Path Choice

**Masterwork Tools:** Maximize tool efficiency and ore gathering
**Forgeborn Endurance:** Maximize survival and resilience (health and armor)
**Champion:** Requires both paths for passive tool repair
